# Sahil Vachhani Portfolio

Personal portfolio website for Sahil Vachhani, focused on data science, machine learning, NLP, and analytics work.

Live site: [https://sahilsvachhani.github.io/SahilVachhani_Portfolio/](https://sahilsvachhani.github.io/SahilVachhani_Portfolio/)

## Overview

- Single-page portfolio website
- Dark, modern UI with responsive layout
- Animated gravity-stars background effect
- Sections for About, Projects, Experience, and Contact

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Run Locally

1. Clone this repository.
2. Open `index.html` in your browser.

## Deployment

This site is hosted with GitHub Pages from the `main` branch using the root folder (`/`).
